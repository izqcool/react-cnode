# react-cnode
> 使用react仿照[noder](https://github.com/soliury/noder-react-native)写的一个移动端项目,调用[cnoe提供的api](https://cnodejs.org/api);

## install
```
git clone https://github.com/zuoq/react-cnode.git
cd react-cnode
npm install || yarn install
npm start || yarn start
```

1. 登录/退出

3. 查看评论
4. 发表评论
5. 回复他人评论
6. 点赞
7. 发表文章
8. 查看已读/未读消息
